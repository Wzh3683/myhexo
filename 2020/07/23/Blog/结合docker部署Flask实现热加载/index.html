在Oracle VM VirtualBox中设置共享文件夹

  1. 我们先打开我们的Oracle VM VirtualBox。右键选择设置。
  2. 选择共享文件夹，然后添加我们的文件夹。
  3. 选择文件加路径，选择自动挂载，固定分配。

查看文件夹是否存在

  1. 进入我们的默认镜像
  2. 进入根目录
  3. 查看我们的共享文件夹
  4. 这时候我们发现我们的文件夹已经映射到默认镜像了。
  5. 退出默认镜像
